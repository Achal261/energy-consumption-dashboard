Energy Consumption Dashboard — Power BI


An interactive Power BI dashboard built during the Edunet Foundation Virtual Internship to analyze energy consumption patterns across regions, sectors, and time periods.




 Project Overview

This project involves end-to-end data analysis of an energy consumption dataset sourced from Kaggle. The goal was to transform raw energy data into a clean, interactive dashboard that helps stakeholders understand usage trends, identify peak consumption periods, and support cost and efficiency decisions.

 Project Workflow

1. Data Collection


Downloaded energy consumption dataset from Kaggle
Reviewed structure, columns, and data types before processing


2. Data Cleaning (Power Query)


Handled missing/null values
Corrected data types for date and numeric columns
Removed duplicates and irrelevant columns
Standardized region and sector labels for consistency


3. DAX Measures & KPIs

Key measures created using DAX:


Total Energy Consumption
Average Consumption per Region
Month-over-Month Growth %
Cost Efficiency Index
Peak Consumption Period


4. Dashboard Features


📌 KPI Cards — Total consumption, average usage, cost efficiency
📊 Bar/Column Charts — Consumption comparison across regions and sectors
📈 Line Charts — Energy usage trends over time
🍩 Pie/Donut Charts — Sector-wise distribution of consumption
🗺️ Map Visual — Geographic breakdown of energy usage
🎛️ Slicers/Filters — Dynamic filtering by time period, region, and sector



 Key Insights


Identified peak energy consumption periods and seasonal usage patterns
Compared energy usage across regions and sectors to highlight high-consumption areas
Supported cost and efficiency analysis through calculated KPI metrics
Enabled stakeholders to drill down into specific time periods and regions using interactive slicers
